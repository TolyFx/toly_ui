## 一、核心颜色模型对比

### 1.1 HSV（HSB）模型

| 维度 | 说明 |
|------|------|
| **全称** | Hue-Saturation-Value（色相-饱和度-明度） |
| **别名** | HSB（Brightness） |
| **色相 H** | 0°红 → 60°黄 → 120°绿 → 180°青 → 240°蓝 → 300°品红 |
| **饱和度 S** | 0 = 灰色，1 = 最鲜艳纯色 |
| **明度 V** | 0 = 黑色，1 = 纯色（最大亮度） |
| **几何体** | 倒圆锥体（H = 角度，S = 半径，V = 高度） |

**经典拾色器 UI**（Photoshop / GIMP）：

```
┌──────────────────────────┐  ┌───┐
│                          │  │ 红 │
│      S-V 方形面板         │  │    │
│   ← 饱和度 S（横轴）       │  │ 橙 │ 色
│   ↑ 明度 V（纵轴）         │  │    │ 相
│   左上: 白（S=0 V=1）     │  │ 黄 │ 条
│   右上: 纯色（S=1 V=1）   │  │    │
│   左下: 黑（S=0 V=0）     │  │ 绿 │
│   右下: 暗纯色（S=1 V=0） │  │    │
│                          │  │ ... │
└──────────────────────────┘  └───┘
```

**与 HSL 的关键差异：**
- HSV 的 V 轴底部是黑色（无光），纯色的 V=1
- HSL 的 L 轴底部黑色、顶部白色，纯色的 L=0.5（中间）
- HSV 描述「从黑色到纯色」（颜料混合方向）
- HSL 描述「从黑色到纯色再到白色」（物理亮度方向）

---

### 1.2 HSL 模型

| 维度 | 说明 |
|------|------|
| **全称** | Hue-Saturation-Lightness（色相-饱和度-亮度） |
| **色相 H** | 与 HSV 相同，0°~360° |
| **饱和度 S** | 0 = 灰色，1 = 最鲜艳 |
| **亮度 L** | 0 = 黑色，0.5 = 纯色，1 = 白色 |
| **几何体** | 双圆锥体（上下对称） |

**典型 UI**（CSS 调色工具）：

```
┌──────────────────────────┐  ┌───┐
│ 白色 ─────────→ 高饱和    │  │ 红 │
│  (L=1，S递增)             │  │    │
│ ─ ─ ─ L=0.5 纯色中线 ─ ─ │  │ 橙 │
│ 黑色 ─────────→ 暗纯色    │  │    │
│  (L=0，S递减)             │  │ ... │
└──────────────────────────┘  └───┘
```

**实现要点：**
- 先画水平饱和度渐变（灰色→纯色，L=0.5）
- 上半部叠加白色渐变 → L: 0.5→1
- 下半部叠加黑色渐变（multiply） → L: 0.5→0

**适用场景：** CSS 颜色系统、Web 前端、设计师配色工具。

---

### 1.3 RGB 模型

| 维度 | 说明 |
|------|------|
| **全称** | Red-Green-Blue（红-绿-蓝加色模型） |
| **通道** | R: 0~255，G: 0~255，B: 0~255 |
| **几何体** | 正方体（三维笛卡尔坐标） |
| **特例** | (0,0,0)=黑，(255,255,255)=白 |

**典型 UI（三通道滑条）：**

```
┌──────────────────────────────┐
│ R ■══════════════■─────■───■ │  0 ──────────→ 255
│   滑条背景从 (0,G,B) 渐变到 (255,G,B)  │
│                              │
│ G ■────■══════════■─────■───■ │  背景联动 R,B 值
│                              │
│ B ■────────■══════■─────■───■ │  背景联动 R,G 值
│                              │
│  ● 预览色块         R: 128   │
│                     G: 200   │
│                     B: 64    │
└──────────────────────────────┘
```

**适用场景：** 程序员精确调色、需要数字输入的场景。

## 二、拾色器 UI 形态分类

### 2.1 方形面板 + 色相条（最主流）

| 模式 | 方形面板内容 | 色相条 | 代表产品 |
|------|-------------|--------|----------|
| **HSV** | S-V 面板 | 垂直色相条 | Photoshop, GIMP, Figma |
| **HSL** | S-L 面板 | 垂直色相条 | CSS 调试工具 |

**占空间：** 中 ~300×200  | **直觉性：** ⭐⭐⭐⭐⭐

---

### 2.2 圆形色相环 + 明度滑条

```
           ╭──────────╮
         ╱  ╲  红    ╱  ╲
       │ 品红      黄    │      ┌───┐
       │    ╭──╮  ●    │      │ 明 │
       │ 蓝 │中│   青   │      │ 度 │
       │    ╰──╯       │      │ 滑 │
         ╲  ╱  绿    ╲  ╱      │ 条 │
           ╰──────────╯        └───┘
        角度=色相  半径=饱和度
```

| 优势 | 劣势 |
|------|------|
| 色相选取直观 | 空间占用大 |
| 视觉吸引力强 | 饱和度控制不精确 |
| 设计感强 | 移动端适配不便 |

**代表产品：** HoloColorPicker, macOS 系统取色器

---

### 2.3 色相环 + 内嵌方形（复合模式）

```
      ╭────────────────╮
    ╱ ╲              ╱ ╲
   │   ╭─────────┐    │
   │   │  S-V    │    │    外环 = 色相
   │   │  面板   │    │    内方 = 饱和度+明度
   │   ╰─────────╯    │
    ╲ ╱              ╲ ╱
      ╰────────────────╯
```

**代表：** flutter_colorpicker 的 `ColorPickerHueRing`, `HueRingPicker`

---

### 2.4 条形拾色器（双滑条，最省空间）

```
┌──────────────────────────┐
│ ■══════════════════════■ │  色相滑条
│ ■══════════════■───────── │  S/V 或 S/L 滑条
└──────────────────────────┘
```

| 优势 | 劣势 |
|------|------|
| 占用空间极小 | 选色不够直观 |
| 适合实时预览场景 | 需要在两个维度间切换 |

**代表产品：** 网易云音乐换肤、移动端轻量拾色器

---

### 2.5 HEX 文本输入

```
┌──────────────────────────┐
│ [# FF 4D 79 A8  ]  ✓    │
│  输入 6/8 位 16 进制值    │
│  支持复制粘贴自动解析      │
└──────────────────────────┘
```

**特点：** 空间占用最小、精确复制粘贴、通常作为辅助模式。

---

│ ■══════════════════════■───── │  透明(α=0) → 不透明(α=1)

## 三、业界主要工具的实现对比

| 工具/库 | 支持的模式 | 主要 UI 形态 |
|---------|-----------|-------------|
| **Photoshop** | HSV + RGB + HEX + CMYK | 方形 S-V 面板 + 色相条 |
| **Figma** | HSV + RGB + HEX + HSL | 方形面板 + 色相条 |
| **colorpicker.dev** | OKLCH, HWB, HSL, HSV, RGB, HEX | 共用色相环 + 各模式独立滑条 |
| **flutter_colorpicker** | HSV, HSL, RGB, Material | 色相环+方形 或 独立滑条 |
| **Chrome DevTools** | HEX + HSL + RGB | 色相条 + S/L 方形面板 |
| **HoloColorPicker** | HSV | 圆形色相环 + S/V/Alpha 三滑条 |
| **macOS 取色器** | RGB + HSB + CMYK + HEX | 色相轮 + 方形面板 |
| **网易云音乐** | HSB 简化 | 条形双滑条 |
